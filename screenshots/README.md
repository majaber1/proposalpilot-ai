# Screenshots

This folder contains screenshots of the ProposalPilot AI application.

## Planned Screenshots

- `01-landing-page.png` - Landing page with features and pricing
- `02-login-page.png` - Login page
- `03-dashboard.png` - Main dashboard with stats
- `04-upload-rfp.png` - RFP upload page
- `05-analysis-results.png` - AI analysis results tabs
- `06-export-markdown.png` - Exported proposal in markdown

## Adding Screenshots

After running the app with `docker compose up -d`:

1. Open http://localhost:3000
2. Take screenshots at each page
3. Save them here with the naming convention above

These screenshots help potential users understand the app before installing.
